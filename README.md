# Soberat (Frontend)

> A cross-platform mobile application that calculates Blood Alcohol Content (BAC) from biometric inputs and estimates exactly when a user will be sober. 

## 🚀 Overview

This repository contains the frontend client for Soberat. Designed as a hybrid mobile application, it provides a fast, responsive, and intuitive user experience across devices. The application features a dynamic UI enriched with custom animations and interactive dashboards to help users track their consumption patterns over time.

*(Note: This frontend is built to integrate seamlessly with a custom PHP/MySQL backend via a RESTful API).*

### Key Features
* **Real-Time BAC Calculation:** Processes biometric inputs to accurately estimate Blood Alcohol Content.
* **Sobriety Countdown:** Provides users with a clear, algorithmic estimation of when they will be completely sober.
* **Dynamic Dashboards:** Enables historical data tracking to monitor consumption patterns and beverage preferences.
* **Optimized Mobile UX:** Utilizes responsive design and custom animations to deliver immediate, satisfying user feedback and improve retention.
* **Cross-Platform Delivery:** Built as a hybrid application to ensure a consistent experience across different mobile operating systems.

## 🛠️ Tech Stack

* **Mobile Framework:** Ionic
* **Core Framework:** Angular
* **Styling:** Sass
* **Animations:** Lottie

## 🚦 Getting Started

Follow these instructions to get a local copy of the project up and running on your machine.

### Prerequisites
* [Node.js](https://nodejs.org/)
* [Ionic CLI](https://ionicframework.com/docs/cli) (`npm install -g @ionic/cli`)

### Build & Run Instructions

1. Clone the repository  
git clone https://github.com/weatherr/Soberat-Front-end.git

2. Navigate into the directory  
cd Soberat-Front-end

3. Install the dependencies  
npm install

4. Start the development server  
ionic serve
